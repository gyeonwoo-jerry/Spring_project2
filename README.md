1. 프로젝트 간략한 설명
User (사용자): 사용자 데이터를 관리합니다.
Schedule (일정): 사용자가 일정을 생성하고 수정할 수 있도록 합니다.
Comment (댓글): 사용자가 특정 일정에 댓글을 남길 수 있도록 합니다.
User and Schedule (사용자와 일정): 사용자를 일정에 연결하는 역할을 합니다.
